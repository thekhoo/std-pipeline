## Project Overview

std-pipeline is a Python project managed with [uv](https://docs.astral.sh/uv/). Python 3.14+ is required.

## Commands

- **Run**: `uv run main.py`
- **Add dependency**: `uv add <package>`
- **Sync environment**: `uv sync`
