# std-pipeline

## Prerequisites

- **Python 3.14+** — install via [python.org](https://www.python.org/downloads/) or a version manager like [pyenv](https://github.com/pyenv/pyenv)
- **uv** — install from [docs.astral.sh/uv](https://docs.astral.sh/uv/getting-started/installation/)

## Setup

```sh
# Install Python 3.14 (if using pyenv)
pyenv install 3.14

# Sync the project environment
uv sync
```

## Usage

```sh
uv run main.py
```
